# AI-Powered-Train-Assistant-for-Ireland
AI assistant for Irish rail passengers, providing ticketing, accessibility and other journey-related information. Built with RAG and LangChain for retrieval from relevant sources. GPT-4.1-mini generates grounded responses. GPT-4o-mini-TTS reads messages aloud. Gradio provides a user-friendly interface.

To run the project, please open and execute **AI_Powered_Train_Assist.ipynb**. The **Knowledge-base** folder contains publicly disclosed information on train services in Ireland and serves as the knowledge source for the RAG pipeline. A project demo is provided as an **Project_Demo_AI_Powered_Train_Assistant_for_Ireland.mp4** file.
